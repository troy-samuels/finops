# 09. SDLC And Supply Chain Controls

## Objective
Enforce auditable delivery controls for secure and reliable releases.

## Execution Checklist
- Enforce CI quality gates (typecheck, lint, test, build, dependency audit).
- Define branch protection and review requirements.
- Add secret scanning and dependency vulnerability checks.
- Define migration safety and rollback procedure.

## Evidence
- CI workflow and passing run links
- Release checklist
- Rollback runbook

## Exit Criteria
- Every release path is gated and traceable.
